# DCIT-313-Group-Tech-folks-SmartBotanica-Expert-System

SmartBotanica is a rule-based Expert System designed to assist gardeners, farmers and horticulture professionals in diagnosing plant diseases and determining appropriate care strategies. By encoding expert botanical knowledge into a structured knowledge base and applying logical inference the system bridges the gap between domain expertise and everyday plant care.

The system accepts user-supplied observations including plant species, visible symptoms and environmental conditions and reasons over these inputs using a set of IF–THEN production rules derived from expert botanical research. The result is a clear, actionable diagnosis accompanied by targeted care recommendations.
